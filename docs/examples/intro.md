# Examples

This section contains a collection of Jupyter notebooks demonstrating the capabilities of OpenGeo for various remote sensing tasks.

## Available Examples

* **[NDVI Analysis](landsat8_ndvi_analysis.ipynb)**: Learn how to calculate the Normalized Difference Vegetation Index using Landsat 8 data.
* **[Sentinel-2 Exploration](sentinel2_analysis_element84.ipynb)**: Accessing and visualizing Sentinel-2 imagery from Element84.
* **[Landsat 8 USGS](landsat_analysis_usgs.ipynb)**: Working with Landsat 8 collections from the USGS STAC API.
* **[MODIS NASA](modis_analysis_nasa.ipynb)**: Accessing land surface temperature or other MODIS products.
* **[SAR Analysis](sar_analysis_umbra_capella.ipynb)**: Working with Synthetic Aperture Radar data from Umbra and Capella.

!!! tip "Running Notebooks"
    You can download these notebooks from the GitHub repository and run them locally using Jupyter Lab or VS Code.
