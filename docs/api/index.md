# API Reference

The OpenGeo API is structured to mirror the Earth Engine Python API. Below is the documentation for the core classes.

## Core Modules

* **[`og.Image`](image.md)**: Operations on single multi-band raster images.
* **[`og.ImageCollection`](image_collection.md)**: Operations on groups of raster images.
* **[`og.FeatureCollection`](feature_collection.md)**: Operations on vector datasets.
* **[`og.Geometry`](geometry.md)**: Defining spatial regions.
* **[`og.Map`](map.md)**: Interactive visualization.
